# IPTV Playlist

This repository contains my personal M3U IPTV playlist for streaming live TV channels.

## 📌 How to Use

1. Copy the **Raw M3U link** from this repository.
2. Open your preferred IPTV player (e.g., VLC, IPTV Smarters, Perfect Player).
3. Paste the link in the app to start streaming the channels.

## ⚠️ Notes

- This playlist is for personal use only.
- If it contains paid subscription links, please **do not share publicly**.
- Make sure your IPTV player supports M3U format.

## 🔗 Raw M3U Link

You can get the raw link by clicking the `Raw` button on `playlist.m3u`.
